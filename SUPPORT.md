# Support

## What This Project Provides
- Open source blueprint and implementation guidance for Club OS.
- Community-driven support via issues and discussions.

## What This Project Does Not Provide
- Guaranteed SLA-based support.
- Custom feature delivery on request.
- Managed operations for third-party deployments.

## Getting Help
1. Search existing docs in `/docs`.
2. Search existing issues/discussions.
3. Open a new issue with reproducible details and environment context.

## Good Issue Reports Include
- expected behavior
- actual behavior
- logs/errors
- steps to reproduce
- deployment profile (managed or self-host/cloud-portable)
